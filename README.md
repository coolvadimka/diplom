# Интерактивный калькулятор многозначной логики

Веб-приложение предназначено для выполнения вычислений в пользовательских k-значных логических системах. Система позволяет вводить логические выражения, задавать количество значений истинности и переменных, строить таблицы истинности, получать основные формы логических функций и экспортировать результаты в Excel.

Дополнительно в приложении реализованы учебные материалы, модуль тестирования и разграничение доступа пользователей по ролям: студент, преподаватель и администратор.

## Основные возможности

- вычисление выражений k-значной логики;
- построение таблиц истинности;
- отображение промежуточных этапов вычислений;
- формирование основных форм логических функций;
- экспорт результатов в Excel;
- регистрация и авторизация пользователей;
- работа с учебными материалами;
- прохождение тестирований;
- управление пользователями и ролями.

## Технологии

- Django

- HTML
- CSS

## Назначение

Приложение разработано для автоматизации вычислений в многозначной логике и может использоваться в учебном процессе при изучении дискретной математики, математической логики и k-значных логических функций.
